Event Types Being Tracked

This demo page tracks the following event types:

  • page_view - Automatically tracked on page load and navigation
  • click - Tracked for all clicks on the page
  • button_click - Specifically tracked for button elements (with button metadata)
  • form_submit - Tracked when forms are submitted (submit the form above)
  • input_change - Tracked when input fields change value (type in form fields above)
  • input_focus - Tracked when inputs receive focus (click on form fields above)
  • scroll - Tracked when page is scrolled (scroll this page)
  • image_view - Tracked when images load (images in gallery above)

Tip: Interact with different elements above to generate various event types. Then check the dashboard to see all tracked events with their metadata.
